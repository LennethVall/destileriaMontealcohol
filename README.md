# 🍶 Destilería Montealcohol

Aplicación de gestión para una tienda de licores artesanales.  
Permite administrar productos, clientes, ventas y operaciones internas mediante una interfaz desarrollada en Java y conectada a una base de datos MySQL.

---

## 📝 Descripción
Destilería Montealcohol es un sistema de gestión diseñado para facilitar el control de inventario, la administración de productos artesanales y el registro de operaciones comerciales.  
Incluye paneles gráficos, validaciones, acceso a base de datos y pruebas unitarias.

---

## 🧰 Tecnologías utilizadas
- Java 8  
- Swing (interfaz gráfica)  
- MySQL  
- JDBC (MySQL Connector Java 8.0.15)  
- Mockito (mockito-core 5.12.0)  
- Byte Buddy 1.14.12  
- MySQL Mockito (para simulación de conexiones)  
- IDEs compatibles: Eclipse / IntelliJ IDEA / VS Code  

---

## 📚 Dependencias
Este proyecto utiliza las siguientes librerías externas:

- **mysql-connector-java-8.0.15.jar**  
  Conector JDBC para la comunicación con MySQL.

- **mockito-core-5.12.0.jar**  
  Framework para pruebas unitarias.

- **byte-buddy-1.14.12.jar**  
  Librería utilizada por Mockito para generar clases dinámicas.

- **mysql-mockito**  
  Utilidad para simular comportamientos de MySQL en tests.

**Añadir los .jar al proyecto:**
- **Eclipse:**  
  `Project → Properties → Java Build Path → Add External JARs`
- **IntelliJ:**  
  `File → Project Structure → Modules → Dependencies → + → JARs`
